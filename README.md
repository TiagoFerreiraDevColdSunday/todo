# TODO Program

## Introduction
This is my first Rust project.

The feature is simple: on your command line you can add your todo notes to your computer.

## How to install it

Makefile commands:

Full instalation:
```shell
make install_todo
```

Install cargo:
```shell
make install_cargo
```

## Usage

### Add:
```shell
todo add "A very Bad example" "I need to do a very bad example" "2025-02-20" 
```
